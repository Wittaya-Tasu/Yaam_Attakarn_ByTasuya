# ยามอัฐกาล By Tasuya

เว็บแอปแบบ Mobile-first สำหรับคำนวณยามอัฐกาลตามวันและเวลา

## คุณสมบัติ
- คำนวณวันยามใหม่เวลา 06:01 น.
- เวลา 00:00 เป็นช่วงปลายของยาม 22:31–00:00 ของวันก่อน
- แสดงยามกลางวันและกลางคืนอย่างละ 8 ช่วง
- แบ่งแต่ละยามเป็น ต้น / กลาง / ปลาย พร้อมเวลาเริ่ม
- เลือกดู 4 หมวด: เรื่องราว เจ็บไข้ หาของ เดินทาง
- ค่าเริ่มต้นเป็นหมวดเดินทาง
- สีเขียว / เหลือง / แดงตามความหมาย
- Dark mode, PWA, Offline cache และอัปเดตเวลาปัจจุบันอัตโนมัติ

## วิธีขึ้น GitHub Pages
1. สร้าง Repository ใหม่
2. อัปโหลดไฟล์และโฟลเดอร์ทั้งหมดไว้ที่หน้าหลักของ Repository
3. ไปที่ Settings → Pages
4. เลือก Deploy from a branch
5. เลือก Branch `main` และ Folder `/ (root)`
6. กด Save และรอระบบสร้างเว็บไซต์

## หมายเหตุ
